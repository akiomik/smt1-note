# 悪魔

悪魔のデータ一覧。メモリ上の `0x07CBFA` から、1体当たり24バイト、合計310体定義されている模様。

:::info NOTE
- ALIGN (属性) 列は0〜105がLAW、106〜144がNEUTRAL、145〜255がCHAOS。
- アイテム列は [ドロップアイテムのID](/docs/item) だが、 `0x00` はアタックナイフではなくドロップなしとして扱われている。
- DARK以外の悪魔にマグネタイト係数が設定されている理由は要調査。
- 出現しない悪魔に経験値&ドロップ率係数が設定されている理由は要調査。
:::
